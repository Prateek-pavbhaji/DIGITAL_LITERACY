## 📌 Project Overview
This project focuses on building a **Digital Literacy Portfolio** to develop essential skills required in today’s digital world.It covers creating a professional online presence, understanding digital tools, maintaining proper communication, and ensuring cyber safety.

The project is divided into five modules, each addressing a key area of digital literacy such as awareness, portfolio building, collaboration tools, communication etiquette, and cybercrime prevention.

### 🔐 Task 5: Cybercrime Awareness
* **Learned about:**
  **UPI Fraud:** Understanding how fraudsters use "Request Money" scams and social engineering to steal funds.
  **Scam Mechanics:** Realizing that entering a UPI PIN is only for sending money, never for receiving it.
  **Key takeaway:** Sophisticated psychological manipulation is often more dangerous than technical hacking.
  **Safety Practices:** * Activate Multi-Factor Authentication (MFA) on all critical accounts.
    Strictly audit app permissions like "Background Location" and "Contacts".
  **Reporting Channel:** National Cyber Crime Portal (cybercrime.gov.in) and Helpline **1930**.
